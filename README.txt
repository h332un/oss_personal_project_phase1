1. 구현 목표
본 프로젝트의 목적은 업무를 효율적으로 관리하기 위한 To-Do-List를 만드는 것입니다.

2. 구현 기능
1) 회원가입, 로그인
2) 완료된 업무, 해야 하는 업무 한 눈에 확인 및 진행 상태를 프로그레스바로 확인
3) 업무 추가, 삭제, 수정
4) 업무마다 이름, 기한, 상단 고정 여부, 월/주/일마다 주기적으로 있는 업무인지 지정할 수 있는 옵션 제공

3. 실행 방법
1) JDK 다운로드
2) 'javac' 명령어로 .class 파일 생성
3) 'java' 명령어로 실행

4. 코드 설명
1) A_MainApplication 클래스는 사용자가 이미 가입 정보가 있는지 확인하고 로그인 또는 가입 화면을 표시합니다.
2) B_SignupScreen 클래스는 Signup Screen을 나타내는 클래스입니다. 사용자가 필요한 정보를 입력하고 회원가입을 진행할 수 있습니다.
3) C_LoginScreen 클래스는 Login Screen을 나타내는 클래스입니다. 사용자가 저장된 정보로 로그인할 수 있습니다.
4) D_Task 클래스는 Task 정보를 나타내는 클래스입니다. 할 일의 제목, 기한, 상단 고정 여부, 주기적 여부, 완료 여부 등의 정보를 포함합니다.
5) E1_TaskOverviewScreen 클래스는 Task Overview Screen을 나타내는 화면 클래스입니다. 사용자의 완료되지 않은 할 일과 완료된 할 일을 구분하여 표시합니다.
현재 task의 크기가 너무 작게 표시되어서 보이지 않는다는 문제가 있습니다. 시간 내에 해결하지 못해서 일단 제출하였습니다.
6) E2_TaskDetailScreen 클래스는 Task Detail Screen을 나타내는 화면 클래스입니다. 할 일의 상세 정보를 수정할 수 있습니다.
